# 全量重写阶段计划

## 目标

把 `learn-claude-code` 的教学型能力重写为可长期维护、可扩展、可验证的工程化实现。

## 阶段拆分

1. 阶段-01 基础内核
   - 核心循环、工具注册、阶段日志机制
2. 阶段-02 任务系统
   - 任务拆解、依赖关系、持久化状态
3. 阶段-03 背景执行
   - 异步任务、完成通知、失败重试
4. 阶段-04 多代理协同
   - 子代理调度、消息协议、状态同步
5. 阶段-05 上下文治理
   - 压缩、分层记忆、会话恢复
6. 阶段-06 安全与权限
   - 工具白名单、命令限制、审计追踪
7. 阶段-07 可观测性
   - 指标、日志、阶段报告自动生成
8. 阶段-08 生产化收口
   - CLI/API、部署脚本、回归测试矩阵

## 完成标准

- 每阶段必须有：
  - 可运行代码
  - 可复现验证（测试/命令）
  - 阶段记录文档
